Soy desarrolladora full-stack apasionada por el código limpio y las interfaces cuidadas. Disfruto resolviendo problemas reales con soluciones elegantes.

Actualmente curso **DAM** y compagino los estudios con mi labor como **monitora voluntaria**, apoyando iniciativas como **STEM Girls** para que haya más mujeres en tecnología.
